# Binom Magic

English version: [README.md](C:/Users/Mikhail/binom-magic-extention/README.md)

Chrome MV3 расширение для страниц отчетов Binom. Показывает метрики источника в hover-overlay без изменения таблицы отчета.

## Что умеет

- Удержание `Ctrl` над поддерживаемой строкой открывает overlay
- Двойное нажатие `Ctrl` закрепляет или открепляет overlay
- Через контекстное меню можно сделать prefetch видимых строк
- `Alt+Shift+C` копирует debug IDs
- Из закрепленного overlay можно останавливать и запускать кампании и тизеры
- Из закрепленного overlay можно перезаписывать расход в Binom текущим spend из источника

## Для чего нужно

Расширение сделано под связку Binom + AdsKeeper:

- читает контекст отчета со страницы Binom
- определяет связанную сущность на стороне источника
- запрашивает метрики в фоне
- показывает их сразу по hover
- дает быстрые операционные действия по кампаниям и тизерам из закрепленного overlay

## Структура проекта

- `extension/` - unpacked Chrome extension
- `docs/` - спецификации и продуктовые заметки

## Локальный запуск

1. Открой `chrome://extensions/`
2. Включи `Developer mode`
3. Нажми `Load unpacked`
4. Выбери `extension/`
5. Открой настройки расширения
6. Заполни свои значения:
   - URL Binom tracker, например `https://your-binom-tracker.example.com`
   - API key трекера Binom
   - credentials источника: `idAuth`, API token и при необходимости `API base URL`

## Важно

- Расширение рассчитано на private/internal usage.
- Оно использует прямые API credentials, поэтому токены трекера и источника нужно считать чувствительными.
- Для командного или production-использования безопаснее backend/proxy слой, чем хранение сильных токенов в браузере.
